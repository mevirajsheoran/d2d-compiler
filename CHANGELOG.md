# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [3.0.0] — 2025

### Added

- **Design Brief** — Structured form capturing page type, industry, tone, brand name, and tagline
- **Industry content engine** — 10 industry-specific content banks with contextual copy
- **Multi-variation generation** — 3 simultaneous design variations per wireframe, <300ms
- **Smart suggestions** — 30 industry × tone mappings for automatic preset recommendations
- **Generation page** — Dedicated page with responsive preview, export options, and style switching
- **Brand permeation** — Brand name and tagline woven into nav, hero, footer, and CTAs
- **Force save** — Direct Convex mutation before page navigation to prevent data loss
- **Hash-based generation cache** — Skip regeneration for identical inputs

## [2.0.0] — 2025

### Added

- **Deterministic generation pipeline** — 7-phase client-side engine (extractor → classifier → architect → enhancer → stylist → builder → display)
- **Shape classification** — 17 semantic UI roles inferred from geometry and type
- **Layout intelligence** — Automatic row/column detection, per-gap spacing, containment hierarchy
- **8 enhancement rules** — Auto-centering, grid detection, hero heading, sticky nav, card wrapping, footer detection
- **Tailwind CSS output** — Responsive, interactive, semantically structured React components
- **Live preview** — Sandpack-powered preview with code editing
- **Zero-cost generation** — All processing in the browser, no API calls

## [1.0.0] — 2025

### Added

- **Infinite canvas** — 10 drawing tools (select, pan, pen, rectangle, ellipse, arrow, line, text, frame, eraser)
- **Project management** — Create, delete, duplicate projects with auto-incrementing names
- **Auto-save** — Every 30 seconds with retry logic and visual status indicator
- **Style guide** — Colors (5 categories), typography (Google Fonts), mood board (up to 5 images)
- **Authentication** — Google OAuth and email/password via Convex Auth
- **Subscription billing** — 3 tiers via Razorpay
- **Dark mode** — System preference detection with manual toggle
