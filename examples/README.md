# Examples

This directory contains sample wireframe data for testing the D2D pipeline.

## Usage

These JSON files can be used with the pipeline tests or loaded into the canvas for development:

```typescript
import {
  LANDING_PAGE_FRAME,
  LANDING_PAGE_SHAPES,
} from "../tests/fixtures/shapes";
```

## Contributing Examples

If you create an interesting wireframe layout, export the shapes JSON and submit a PR to add it here.
